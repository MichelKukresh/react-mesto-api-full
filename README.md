# Проект: Путешествие по России

### Одностраничный сайт, с тематикой путешествия. Есть элементы анимации, есть возможность добавления мест путешествия, отмечать лайками.

###### Создана файловая структура по БЭМ (Nested). Фреймворк React.js. Используются контейнеры flex, grid адаптирован под различные зазрешения и браузеры, express, Node.js, mongoDB.
